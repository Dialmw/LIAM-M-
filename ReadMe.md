 # LIAM M WhatsApp Bot

A modular WhatsApp bot named **LIAM M**, built with Node.js and Baileys, featuring AI, media downloads, and group management.

## Setup (Local)

1. Install Node.js (v16+).
2. Install FFmpeg:
